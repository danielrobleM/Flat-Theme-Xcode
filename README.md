# Flat Theme For Xcode

Xcode Theme built by Flat colors

## Installation

### Use [Alcatraz](http://alcatraz.io/)
Find 'Flat Theme Xcode'

### Manual
1. Copy .dvtcolortheme files to ~/Library/Developer/Xcode/UserData/FontAndColorThemes/
2. Restart Xcode
3. Have fun!
